    Code prototypes demo, include ScalaDemo/SparkDemo/DataLoader etc.
    Build project with sbt:
    $ cd ./CodePrototypesDemo/
    $ sbt update gen-idea
    $ sbt clean assembly
    
    Build project with maven:
    $ cd ./CodePrototypesDemo/
    $ mvn clean install -DskipTests
    $ mvn clean package -DskipTests
